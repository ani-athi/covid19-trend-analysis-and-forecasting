# 🦠 COVID-19 Trend Analysis and Forecasting using Prophet

## 📌 Overview

This project analyzes COVID-19 data and forecasts future trends using time-series modeling. In addition to exploratory data analysis, the Prophet model is used to predict the progression of cases over time.

## 🧠 Approach

* Exploratory Data Analysis (EDA)
* Time-Series Forecasting using Prophet

## ⚙️ Workflow

1. Data loading
2. Data cleaning and preprocessing
3. Time-series data preparation
4. Exploratory data analysis
5. Visualization of trends
6. Applying Prophet model for forecasting
7. Generating future predictions
8. Visualizing forecast results

## 📊 Analysis Performed

* Trend of COVID-19 cases over time
* Growth patterns and changes in trends
* Forecasting future case counts using Prophet

## 🔮 Forecasting Model

* Prophet (developed by Meta)
* Handles trend and seasonality in time-series data
* Used to generate future predictions based on historical data

## 📈 Tools & Libraries

* Python
* Pandas
* Matplotlib
* Seaborn
* Prophet

## 📊 Key Insights

* Understanding spread patterns of COVID-19
* Visualizing trends across time
* Predicting future case trends using time-series modeling
